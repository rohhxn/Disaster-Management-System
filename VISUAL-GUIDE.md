### On Your Local Machine

```bash

ssh-keygen -t rsa -b 4096 -f ~/.ssh/dams-key -N ""
cd terraform-demo
cp terraform.tfvars.example terraform.tfvars

# Deploy
terraform init

terraform destroy
```

### On EC2 Instance (After SSH)

```bash

# Create secure .env file
cat > .env << 'EOF'
NODE_ENV=production

DB_PORT=5432
```
